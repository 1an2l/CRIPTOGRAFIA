1. Clonar el repositorio (si aún no lo tienes en tu computadora):

git clone https://github.com/1an2l/CRIPTOGRAFIA.git


2. Entrar al directorio del repositorio:

cd CRIPTOGRAFIA


4. Ver el estado del repositorio (para saber qué archivos han cambiado):

git status


5. Agregar archivos nuevos o modificados para el commit:

Para agregar todos los archivos cambiados

git add .


O para agregar un archivo específico

git add nombre_del_archivo.ext


6. Hacer un commit con un mensaje descriptivo:

git commit -m "Descripción de los cambios realizados"


7. Subir los cambios al repositorio en GitHub:

git push origin main

8. Actualizar tu repositorio local con los cambios del repositorio remoto:

git pull origin main
